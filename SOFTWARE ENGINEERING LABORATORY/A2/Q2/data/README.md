## Dataset Format

Images are to be downloaded in the ``` imgs ``` folder.  

``` annotations.jsonl ``` contains the annotations for the images. The format of the annotations are:

```
{
    "file_name": 
    "url":
    "captions": [ {
            "caption":
        }
        .
        .
    ]
}
```

A sample of annotation file for 10 images are given.
